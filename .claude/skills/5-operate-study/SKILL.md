대화 맥락을 학습 가능한 문서로 변환한다.

## 경로 해석

학습 노트의 저장 경로는 `.claude/docs-file.json`의 `study_root`를 사용한다.
설정 파일이 없거나 `study_root`가 없으면 사용자에게 경로를 물어본 뒤 `.claude/docs-file.json`에 추가한다.

## 출력 계약 (Output Contract)

반드시 아래 산출물을 생성/갱신한다.

1. `{study_root}/{category}/[subcategory/]<topic-slug>.md`

카테고리와 파일명은 아래 규칙에 따라 결정한다.

## 트리거 입력

사용자가 최소 아래를 준 경우 실행한다.

- 스킬 호출 이름
- 기록할 주제 키워드(예: `worktree`, `정규화`, `FastAPI Depends`)

선택 입력:
- 카테고리 지정 (없으면 자동 판별)
- 난이도(`입문`, `중급`)
- 문서 길이(`짧게`, `자세히`)
- 코드 예시 포함 여부

## 맥락 수집 규칙

1. 최근 대화 이력에서 키워드가 나온 구간을 우선 탐색한다.
2. 어떤 문제를 해결하는 과정에서 개념이 나왔는지 `상황`을 2~4줄로 요약한다.
3. 필요하면 워크스페이스 파일(관련 문서/코드)을 추가로 확인한다.
4. 맥락이 불명확하면 질문을 최대 2개만 한다.

## 카테고리 결정 규칙

`{study_root}` 아래 기존 폴더를 스캔하여 적합한 카테고리를 선택한다.

현재 존재하는 카테고리 예시:
- `architecture/` — 아키텍처, 설계 패턴, 동기화 전략
- `FastAPI/` — FastAPI, 백엔드, ORM, DB
- `langchain/` — LangChain, LLM, AI 파이프라인
- `library/` — 라이브러리, 도구, 코드 생성기
- `MonoRepo/` — 모노레포, 빌드 시스템
- `Tauri/` — Tauri, 데스크탑 앱

판별 순서:
1. 사용자가 카테고리를 지정했으면 그대로 사용한다.
2. 기존 카테고리 중 주제와 맞는 것이 있으면 선택한다.
3. 기존 카테고리에 없는 새로운 주제면 사용자에게 확인 후 새 폴더를 생성한다.

하위 주제 폴더가 있는 경우(예: `architecture/State Synchronization/`) 해당 하위 폴더에 저장한다.

## 파일명 규칙

기존 study 폴더의 네이밍 컨벤션을 따른다:

- 날짜 접두사를 붙이지 않는다.
- 주제를 명확히 드러내는 이름을 사용한다.
- 케밥케이스(`kebab-case`)를 기본으로 한다.
- 시리즈 문서는 번호 접두사를 사용한다 (예: `1. 주제명.md`).
- 시행 착오/인시던트 기록은 `incident-` 접두사를 사용한다.

예시:
- `merge-engine-embedding-pipeline-hang.md`
- `incident-20260405-push-pending-conflict-markers.md`
- `1. CRDT Deep Dive — 개념과 구현.md`

## 문서 작성 규칙

학습 노트는 아래 섹션을 고정으로 포함한다.

1. 왜 이 개념이 나왔는가 (대화 맥락)
2. 한 줄 정의
3. 핵심 개념 3~5개
4. 우리 프로젝트 적용 포인트
5. 자주 하는 실수/오해
6. 빠른 체크리스트
7. 다음 학습 단계

작성 원칙:
- YAML frontmatter를 사용하지 않는다 (기존 study 문서 컨벤션).
- `#` 제목으로 시작한다.
- 입문자가 읽어도 이해되게 짧은 문장으로 쓴다.
- 추상 설명보다 현재 프로젝트 예시를 우선한다.
- 사실/해석을 구분한다.
- 모르는 내용은 단정하지 않고 `[확인 필요]`로 표시한다.
- 시리즈 문서는 상단에 시리즈 안내 링크를 넣는다.

## 기존 문서 확인 규칙

- 같은 카테고리에 관련 주제 문서가 이미 있으면 새 문서를 만들기보다 기존 문서 하단에 `## 업데이트 로그`를 추가한다.
- 기존 문서와 관련이 있지만 독립 주제면 새 문서를 만들되, 기존 문서에 상호 링크를 넣는다.

## 완료 조건

- 학습 노트가 `{study_root}/{category}/`에 저장됨
- 프로젝트 적용 포인트가 최소 2개 이상 기록됨
- 기존 관련 문서와의 연결이 확인됨
- 다음 학습 액션이 최소 1개 이상 제시됨
